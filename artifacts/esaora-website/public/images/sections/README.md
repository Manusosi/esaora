# Section Background Images

Used as background/visual elements for specific page sections.

## Required Files

| Filename | Section | Usage | Ideal Content |
|---|---|---|---|
| `quote-bg.jpg` | Mission Quote Section (VideoSection) | Full-width background behind the blockquote | Ocean waves, aerial coastal shot |
| `blueprint-bg.jpg` | Blueprint Section (optional texture) | Subtle background texture | Ocean texture, topographic lines |
| `cta-bg.jpg` | CTA Section (optional) | Background behind Fund/Join/Share cards | Dark coastal aerial |

## Pillar Card Images (for Blueprint section — optional enhancement)

| Filename | Pillar |
|---|---|
| `pillar-wash.jpg` | WASH — Water & Sanitation |
| `pillar-climate.jpg` | Climate Action |
| `pillar-blue-economy.jpg` | Blue Economy |
| `pillar-health.jpg` | Public Health |

## Recommended Size
- Full-width backgrounds: 1920×800px
- Pillar cards: 600×600px (square)
- JPG preferred
