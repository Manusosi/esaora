# Hero Section Images

Drop your hero/banner images here. They are used as the full-screen background of the Hero section.

## Required Files

| Filename | Usage | Recommended Size |
|---|---|---|
| `hero-bg.jpg` | Main hero background (ocean/coastal scene) | 1920×1080px or larger |
| `hero-bg-2.jpg` | Slide 2 (optional — for a future hero slider) | 1920×1080px |
| `hero-bg-3.jpg` | Slide 3 (optional) | 1920×1080px |

## Notes
- Use JPG for photos (smaller file size than PNG)
- Images will be overlaid with a dark navy gradient — darker images work best
- Ideal content: East African coastline, Indian Ocean, coastal communities
- Reference in code via: `/images/hero/hero-bg.jpg`
