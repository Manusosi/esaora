# News & Insights Article Images

Used as thumbnail/banner images for news articles on the homepage News section.

## Required Files

| Filename | Article | Ideal Content |
|---|---|---|
| `news-alliance-launch.jpg` | ESA-ORA Alliance Formally Established | Charter signing, team photo, flags |
| `news-mangrove.jpg` | Mangrove Restoration in Mozambique | Mangroves, planting activity |
| `news-blue-economy.jpg` | Blue Economy Innovation Fund | Fishermen, aquaculture, ocean |

## Naming Convention for Future Articles
Use kebab-case: `news-[short-title].jpg`

## Recommended Size
- 800×500px (16:10 landscape)
- JPG preferred

## Reference in code via
`/images/news/news-alliance-launch.jpg` etc.
