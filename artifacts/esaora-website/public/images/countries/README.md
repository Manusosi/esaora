# Country Section Images

Used in the interactive map section — shown when a user clicks on a country card.

## Required Files (EXACT names — case-sensitive)

| Filename | Country | Ideal Content |
|---|---|---|
| `kenya.jpg` | Kenya | Coastline, Mombasa, marine conservation, WASH |
| `tanzania.jpg` | Tanzania | Zanzibar, seaweed farming, fishing community |
| `mozambique.jpg` | Mozambique | Mangroves, coastline, Beira port area |
| `madagascar.jpg` | Madagascar | Coral reef, biodiversity, coastal village |

## Recommended Size
- 600×400px (landscape/card format)
- JPG preferred for file size

## Reference in code via
`/images/countries/kenya.jpg` etc.
